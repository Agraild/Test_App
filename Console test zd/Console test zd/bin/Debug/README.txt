консольное приложение
необходима авторизация
для авторизации в роли управляющего необходима авторизация с данными:
login:admin password:123
для авторизации в роли обычного сотрудника необходима авторизация с данными:
login:user1 password:312
В роли управляющего возможно добавления задач и новых сотрудников
В роли сотрудника возможен просмотр задач и смена статуса задачи
